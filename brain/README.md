# detail-forge — Project Brain

This is the Obsidian knowledge vault for **detail-forge**, a detail-preserving saturator → clipper → limiter audio plugin (Faust + JUCE 8). It holds all durable project context: architecture, modules, roadmap, operations, decisions, known issues, and a dated history.

## How to open
Open this `brain/` folder as an Obsidian vault (Open folder as vault → select `brain/`). Wikilinks (`[[...]]`) and the numbered Map-of-Content (MOC) structure will resolve.

## Start here
Read **`00-Index/00-Home.md`** — the master hub. It links to everything and carries the start-of-session checklist.

## Keeping it current
Say "update the brain" or run `/update-brain`. After meaningful work: append to `50-Audit-Log/51-Timeline.md`, log decisions in `52-Decision-Log.md`, record bugs/risks in `53-Known-Issues.md`. Conventions live in `00-Index/01-How-To-Use-This-Vault.md`.
